How to Use ZEUS

To start using ZEUS for scanning your web applications for security vulnerabilities, follow the steps outlined below:

  1. Set Up: Ensure ZEUS is correctly installed and configured on your system. Refer to the installation guide if needed.
  2. Target URL: Identify the target URL of the web application you wish to scan.
  3. Start Scan: Use the command line interface to initiate a scan. An example command might be: zeus-scan --target http://example.com --project-folder /path/to/save/results
  4. Review Results: Once the scan is complete, ZEUS generates reports in various formats. Navigate to the project folder to review the findings.
  5. Verification: Manually verify the reported findings to confirm their validity and severity.
  6. Take Action: Based on the findings, take the necessary corrective actions to mitigate any vulnerabilities.

For more detailed information on each step, refer to the corresponding sections in this wiki.
