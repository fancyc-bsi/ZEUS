BSTI Integration

For a GUI interface, BSTI can be used to run ZEUS - to get started, make sure you follow the below instructions:

  • Unzip the 7z archive containing the ZEUS binary and configuration files.
  • Drag and drop the content into your existing BSTI directory.
  • The directory structure should look like this (see below):
BSTI/
├── n2p_ng.py
├── nmb.py
├── bsti.py
├── zeus (ZEUS Binary)
└── src/
