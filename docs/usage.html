Command Line Usage

Zeus can be operated from the command line with the following syntax:

zeus [OPTIONS] --project-folder <PROJECT_FOLDER> <TARGET_URL> --proxy <HTTP_PROXY>

Here are the options you can use with ZEUS:

  • --policy <LOGIN_CONFIG>: Path to the policy YAML file (Optional).
  • --project-folder <PROJECT_FOLDER>: Specifies the directory to save scan data.
  • --proxy <HTTP_PROXY>: Uses a specified HTTP proxy for network requests made by ZEUS.
  • --browser Enables browser mode - opens a firefox instance with HUD enabled.
  • --ajax-scan Enables ajax scan which is slow and resource intensive - but can assist with more modern webapps.
  • --zap-port: Allows you to modify port zap will run on.
  • -V, --version: Displays the version information of ZEUS.
